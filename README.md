# RCS LLC Website

Professional website for Riggs Construction Services LLC - Utah-based pipe fabrication, welding, and industrial services contractor.

## Files Included

- `index.html` - Main website HTML file
- `styles.css` - Complete stylesheet with responsive design
- `script.js` - JavaScript for interactivity and animations
- `logo.png` - RCS LLC logo
- `project1.png` - Project gallery image 1
- `project2.png` - Project gallery image 2
- `project3.png` - Project gallery image 3

## Features

✅ **Fully Responsive Design** - Works on desktop, tablet, and mobile devices
✅ **Modern UI/UX** - Clean, professional design with smooth animations
✅ **Services Showcase** - Detailed presentation of all core services
✅ **Technical Capabilities** - Highlighting welding expertise and standards
✅ **Project Gallery** - Visual showcase of completed work
✅ **Contact Form** - Easy-to-use contact form with email integration
✅ **Mobile Navigation** - Hamburger menu for mobile devices
✅ **Smooth Scrolling** - Enhanced navigation experience
✅ **SEO Optimized** - Proper meta tags and semantic HTML

## Deployment Options

### Option 1: Simple File Hosting
1. Upload all files to your web hosting provider
2. Ensure all files are in the same directory
3. Access via your domain name

### Option 2: Free Hosting Services

**GitHub Pages:**
1. Create a GitHub repository
2. Upload all website files
3. Enable GitHub Pages in repository settings
4. Access at: `https://yourusername.github.io/repository-name`

**Netlify (Recommended):**
1. Create free account at netlify.com
2. Drag and drop the website folder
3. Get instant live URL
4. Connect custom domain (optional)

**Vercel:**
1. Create free account at vercel.com
2. Import your project
3. Automatic deployment with custom domain support

### Option 3: Traditional Web Hosting
Upload to providers like:
- Bluehost
- HostGator
- SiteGround
- GoDaddy

## Customization

### Updating Contact Information
Edit `index.html` lines 260-290 (contact section)

### Changing Colors
Edit `styles.css` lines 11-19 (CSS variables):
- `--primary-red`: Main brand color
- `--primary-navy`: Secondary brand color
- Adjust other colors as needed

### Adding More Services
Edit `index.html` in the services section (lines 55-110)
Copy and paste a service card block and modify content

### Updating Images
Replace the image files (logo.png, project1.png, etc.) with your own
Keep the same filenames or update references in index.html

## Form Functionality

The contact form currently opens the user's default email client with pre-filled information. 

**To implement server-side form processing:**
1. Add PHP/Node.js backend
2. Use services like Formspree, EmailJS, or Web3Forms
3. Integrate with email marketing platforms
